JavaScript

O javascript é uma linguagem de programação que está em ascenção atualmente, compondo boa parte das paginas web, esta linguagem é basicamente os musculos de uma pagina, obviamente é uma forma leiga de dizer. Na pratica o JavaScript tambem conhecido como Js tem como responsabilidade junto do HTML e CSS compor a base do front-end. Quando me refiro que ela pode ser o musculo do site quero dizer que com ela podemos dar mobilidade para objetos em nosso site, fazer calculos e exibir caixas de dialogo, no geral o Js pode fazer toda a parte logica da pagina.
